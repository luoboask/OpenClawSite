## 热门趋势 (5条)

1. **[n8n](https://github.com/n8n-io/n8n)** - 开源工作流自动化工具，新增 AI Agent 节点，支持直接调用 LLM 处理数据

2. **[Activepieces](https://github.com/activepieces/activepieces)** - 新兴的开源 Zapier 替代方案，UI 更现代，支持自托管，社区增长迅速

3. **[Hoarder](https://github.com/hoarder-app/hoarder)** - AI 驱动的书签管理工具，自动打标签和摘要，适合信息收集场景

4. **[Dify](https://github.com/langgenius/dify)** - LLM 应用开发平台，新增 Workflow 功能，可构建复杂的 AI 处理流程

5. **[OpenWebUI](https://github.com/open-webui/open-webui)** -  formerly Ollama WebUI，现已支持多模型，成为自托管 AI 聊天界面的首选

## 自动化玩法 (3个)

### 1. n8n + OpenClaw 联动方案
**用途**: 将 n8n 工作流结果推送到微信/WhatsApp
**核心配置**:
- n8n HTTP Request 节点调用 OpenClaw API
- 使用 OpenClaw 的 message 工具发送通知
- 适合价格监控、库存提醒等场景

### 2. 个人知识库自动化
**工具**: Hoarder + OpenClaw + Memos
**玩法**:
- Hoarder 收集网页并 AI 摘要
- OpenClaw Agent 读取摘要并整理
- 自动归档到个人知识库

### 3. 多 AI 模型投票机制
**用途**: 提高回答准确率
**实现**:
- OpenClaw 同时查询多个模型(Claude/GPT/Gemini)
- 对比结果后输出共识答案
- 适合重要决策场景
