# AI咨询中心

专业的 AI 技术咨询服务网站，提供 OpenClaw 部署、AI 工具开发、自动化解决方案等服务。

## 网站地址

- 本地开发: `http://localhost:3001`

## 技术栈

- **框架**: Next.js 15
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建静态站点
npm run build
```

## 项目结构

```
ai-consulting/
├── app/                   # Next.js App Router
│   ├── page.tsx          # 首页
│   ├── layout.tsx        # 根布局
│   ├── services/         # 服务页面
│   ├── contact/          # 联系页面
│   ├── cases/            # 案例展示
│   ├── blog/             # 技术博客
│   └── about/            # 关于我们
├── components/           # 共享组件
└── public/              # 静态资源
```

## 服务项目

1. **OpenClaw 部署服务** - 专业的自托管 AI 网关部署
2. **AI 工具开发** - 定制化工具开发
3. **自动化工作流** - AI 驱动的业务流程自动化
4. **智能客服系统** - 企业级 AI 客服解决方案
